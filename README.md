# Customer Retail Segmentation Using Machine Learning

## Project Overview

Customer segmentation is an important technique used by retail businesses to understand customer behavior and group customers based on similar characteristics. This project focuses on analyzing customer data and applying machine learning techniques to identify meaningful customer segments.

The project uses data preprocessing, exploratory data analysis (EDA), feature engineering, and machine learning algorithms to classify customers into different groups based on their demographic and spending patterns.

---

## Objective

The main objectives of this project are:

- Analyze customer data to identify purchasing patterns.
- Perform data cleaning and preprocessing.
- Explore customer characteristics using EDA techniques.
- Segment customers into different groups using machine learning.
- Generate insights that can help businesses make better marketing and customer relationship decisions.

---

## Dataset Description

**Dataset Name:** Mall Customers Dataset

The dataset contains information about customers and their spending behavior.

### Features:

| Feature | Description |
|---------|-------------|
| CustomerID | Unique identification number of customers |
| Gender | Gender of the customer |
| Age | Age of the customer |
| Annual Income | Annual income of the customer |
| Spending Score | Score assigned based on customer spending behavior |

## Project Workflow

The project is divided into multiple stages:

### 1. Data Cleaning
- Checking missing values
- Removing duplicate records
- Understanding data structure
- Preparing clean data for analysis

### 2. Exploratory Data Analysis (EDA)
- Statistical analysis of features
- Data visualization
- Distribution analysis
- Identifying patterns and relationships between variables

### 3. Feature Engineering
- Encoding categorical variables
- Selecting important features
- Preparing data for machine learning models

### 4. Data Preprocessing
- Feature scaling using StandardScaler
- Preparing ML-ready dataset

### 5. Customer Segmentation
- Finding optimal number of clusters
- Applying K-Means clustering
- Visualizing customer segments

### 6. Model Comparison
Different machine learning models are analyzed for customer segment prediction.

---

## Machine Learning Models Used

### 1. K-Means Clustering
- Type: Unsupervised Learning
- Purpose: Customer grouping based on similarity in behavior and characteristics.
